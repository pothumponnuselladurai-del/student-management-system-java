Student Management System (Java)

📌 Project Description

This is a simple Student Management System developed using Java.
The program allows users to manage student records through a console-based interface.

It demonstrates basic Object-Oriented Programming (OOP) concepts in Java such as classes, objects, loops, and conditional statements.

🚀 Features

- Add new student details
- View all students
- Search student by roll number
- Delete student record
- Menu-driven console interface

🛠 Technologies Used

- Java
- OOP Concepts
- ArrayList
- Scanner class

💻 How to Run the Program

1. Save the file as:

Main.java

2. Open terminal in the project folder.

3. Compile the program:

javac Main.java

4. Run the program:

java Main
